# Project Rules & Style Guidelines

## UI Card & Alert Design Rule
- **STRICTLY FORBIDDEN**: Never use thick left/side border accent styling (e.g. `border-left: 4px...`, `border-left: 5px...`, `border-left: 6px...` or similar side bar borders) on cards, banners, or alert containers.
- **PREFERRED DESIGN**: Use clean, modern card containers with full soft borders (`border: 1px solid rgba(...)`), subtle box-shadows (`box-shadow: 0 10px 30px rgba(...)`), or top status badges instead.
